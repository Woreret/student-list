# 🎓 Журнал Студентів (Student List)

Вебдодаток для ведення списку студентів, створений для практики роботи з JavaScript DOM та localStorage.

## 🌟 Функціонал додатка

- **Додавання студентів**: Форма приймає прізвище, ім'я, курс (1–6) та бал (0–100). Додано імітацію затримки завантаження мережі при збереженні.
- **Збереження даних**: Список синхронізується з localStorage, тому дані зберігаються після оновлення сторінки або закриття браузера.
- **Фільтрація та пошук**: Реалізовано миттєвий пошук по списку студентів за ім'ям при введенні символів у пошуковий рядок.
- **Видалення записів**: Кожен студент має кнопку видалення, яка прибирає його картку зі списку та сховища.

## 🛠 Технологічний стек і методи

- **HTML5 та CSS3**: Базова розмітка та стилізація інтерфейсу.
- **Vanilla JavaScript (ES6+)**:
  - Логіка побудована на класах (`Student`, `Gradebook`) з використанням приватних полів (`#students`).
  - Для реалізації затримки використано асинхронні функції та проміси (`async/await`, `Promise`).
  - Унікальні ID для записів генеруються через `crypto.randomUUID()`.
  - Валідація введених даних працює через блоки `try...catch`.

## 🚀 Інструкція із запуску

1. Завантажити файли проєкту або клонувати репозиторій.
2. Відкрити файл `index.html` у браузері. Додаткове налаштування сервера не потрібне.
